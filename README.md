# DogCatcher

## Overview
DogCatcher is an interactive arcade-style game built with JavaScript. The player controls a character that collects bones to gain points while avoiding obstacles within a limited time frame.

## Features
- Real-time player movement
- Dynamic obstacle generation
- Score system with increasing difficulty
- Time-based challenge
- User system with login and registration
- Personal and global high scores
- Leaderboard system using localStorage

## Technologies
- JavaScript (Vanilla)
- HTML5
- CSS3
- localStorage API

## Game Objective
Collect as many bones as possible, avoid obstacles, and achieve the highest score before time runs out.

## Scoring System
- Points are gained by collecting bones
- Collision with obstacles affects the game outcome
- Time limit creates challenge
- Tracks personal and global best scores
